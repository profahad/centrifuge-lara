## Centrifuge Laravel Serve

Centrifugo is a scalable real-time messaging server in language-agnostic way. Set up once and forever.

### Features

- `Centrifugo Communicator`

- `Centrifugo JWT Token Generator`

- `Centrifugo Statistics`

- `Centrifugo Presense`

- `Pusher`
